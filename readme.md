# 📖 Biography Page

A simple and responsive **Biography Page** built using **HTML5** and **CSS3** as part of the **µLearn CL Web Development Task**. The project showcases personal information in a clean and beginner-friendly layout while applying fundamental CSS styling concepts.

## ✨ Features

- 👤 Circular profile picture
- 📝 About Me section
- 🎓 Education details
- 💻 Skills displayed as styled tags
- 🔗 Contact links (GitHub, LinkedIn, Email, Instagram)
- 📱 Responsive and minimal design
- 🎨 Clean card-based layout with subtle shadows and rounded corners

## 🛠️ Technologies Used

- HTML5
- CSS3

## 🚀 Getting Started

1. Clone this repository:

   ```bash
   git clone https://github.com/your-username/biography-page.git
   ```

2. Navigate to the project folder:

   ```bash
   cd biography-page
   ```

3. Open `index.html` in your preferred web browser.
